Sonic Visualiser Analysis







Time-frequency analysis is more reflective of the actual changes and duration of notes in a piece. It is easier for human to quickly distnguish pitches and for how long they last at any given point in a song when looking at time-frequency analysis, which might help in decoding song's melody for example. Compared to that, waveform provides some information about piece's loudness level throughout time, but it lacks more features that could help identify the song more precisely. Looking at both the waveform and time-frequency for "Platform 9 and 3 quarters" for example, we can see where there are more dynamic parts of a song in both images, but while waveform will give us this information only, time-frequency analysis can be used to understand which notes are played in those more dynamic moments, and thus decode the melody for a given timeframe.
